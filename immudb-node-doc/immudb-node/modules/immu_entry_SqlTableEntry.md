### binEntryPrefixedKeyToSqlTableEntryPart

▸ **binEntryPrefixedKeyToSqlTableEntryPart**(`b`): `Object`

- prefix: first byte `0x02`
- tag: bytes `Buffer.from('CTL.TABLE.')`
- dbId: UInt32BE,
