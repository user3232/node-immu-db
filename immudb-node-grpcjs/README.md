## Generate grpc client typescript code

```ps1
npx proto-loader-gen-types `
    --longs=Long `
    --enums=String `
    --bytes=Buffer `
    --defaults `
    --arrays `
    --objects `
    --oneofs `
    --json `
    --grpcLib=@grpc/grpc-js `
    --outDir=src `
    schema.proto
```
